A. UX Diagnosis â€” What's Broken

A1. Top 5 Clarity Problems

#ProblemImpactWho it hurts
1 No product shown on landing page. Parents can't see what their child will use. Zero screenshots, zero demo. Parents bounce â€” they don't trust what they can't see. Parents (buyers)
2 Headline doesn't name the outcome. Current copy talks about "AI-powered essay feedback" â€” a feature, not a result. Value prop fails the 10-second test. Parents think "another AI thing." Parents, Students
3 Fabricated social proof. Fake testimonials + inflated claims ("thousands of students", "70% teacher agreement") destroy trust the moment anyone investigates. One skeptical parent shares it in a WhatsApp group and your credibility is gone. Everyone
4 Pricing value unclear. Free = 3/month vs Plus = unlimited. But what does one submission actually deliver? Parents don't know. Low conversion to Plus â€” parents can't calculate ROI vs tuition. Parents
5 Student nav is a junk drawer. 8 items including 3 gamification features (Achievements, Wishlist, Trophy Case) that are adjacent concepts presented as separate destinations. Cognitive overload. Students don't know where to go. Key actions buried. Students

A2. Top 5 Friction Points

#FrictionWhereFix type
1 Registration requires parent to create account AND create first assignment before student sees anything. Too many steps before value. Onboarding Restructure flow
2 No sample essay / demo mode. Both parent and student must commit (register + create assignment + write essay) before seeing ANY AI feedback. That's 10+ minutes to first value. Activation Add instant demo
3 Topics page is a separate nav item but it's only used when creating an assignment. It's a reference page masquerading as a primary destination. Student + Parent nav Merge into assignment creation
4 No progress visibility for parents on dashboard home. Parents log in and seeâ€¦ a list of assignments. Not "how is my child doing?" Parent dashboard Redesign dashboard home
5 No FAQ, no Terms, no Privacy Policy on landing page. Singapore parents are privacy-conscious (PDPA). Missing legal links = red flag. Landing page footer Add content
